# tp-lint

Lint PO files from the Translation Project for common errors.

Built with Python CLI. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install tp-lint
```

### Fedora/RPM
```bash
sudo dnf install tp-lint
```
